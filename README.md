# 江苏省高考成绩分析系统 使用手册

## 一、系统简介

江苏省高考成绩分析系统是一款面向高中年级组的考试成绩分析 Web 应用。系统支持江苏省高考"3+1+2"模式下全部 12 种选科组合的成绩管理，提供从成绩录入、赋分转换到多维度统计分析的全流程功能。教师通过浏览器即可完成全部操作，无需安装客户端软件。

### 系统入口

在浏览器地址栏输入系统地址（如 `http://localhost:8000`），按回车键进入登录页面。

### 页面导航

登录后，页面左侧为导航菜单，分为以下几个区域：

- **概览**：仪表盘，展示系统整体数据概览
- **数据管理**：学生管理、考试管理
- **统计分析**：单次考试分析（年级分析、班级分析、个人分析）、多次考试分析、个人追踪分析
- **成绩计算**：成绩计算与赋分模板管理

## 二、登录与退出

### 登录

1. 在登录页面输入用户名和密码
2. 点击"登录"按钮
3. 默认管理员账户：用户名 `admin`，密码 `admin123`

### 退出

点击页面右上角的"退出"按钮，系统将清除登录状态并跳转至登录页面。

## 三、学生管理

点击左侧导航栏"学生管理"，进入学生管理页面。

### 3.1 查看学生列表

页面默认展示全部学生信息，包括学号、姓名、班级、选科组合、入学年份等。可通过以下方式筛选：

- **按班级筛选**：点击"班级"下拉框，选择目标班级
- **按选科组合筛选**：点击"组合"下拉框，选择目标组合（如"物化生"）
- **按入学年份筛选**：点击"入学年份"下拉框，选择目标年份
- **搜索**：在搜索框中输入学号或姓名关键词，实时过滤结果
- **排序**：点击表头列名，可按该列升序或降序排列

### 3.2 新增学生

1. 点击页面上方"新增学生"按钮
2. 在弹出的表单中填写以下信息：
   - **学号**：必填，6-10 位数字
   - **姓名**：必填
   - **班级**：必填，从下拉框中选择
   - **选科组合**：选填，如"物化生""史政地"等
   - **性别**：选填，"男"或"女"
   - **身份证号**：选填，18 位
   - **入学年份**：选填，如 2023
3. 点击"保存"按钮完成新增

### 3.3 编辑学生

1. 在学生列表中，点击某行右侧的"编辑"按钮
2. 修改信息后点击"保存"

### 3.4 删除学生

- **单个删除**：点击某行右侧的"删除"按钮，确认后删除
- **批量删除**：勾选多个学生前的复选框，点击"批量删除"按钮，确认后删除

### 3.5 批量操作

勾选多个学生后，页面上方会出现批量操作按钮：

- **批量调班**：将选中学生全部转移到目标班级，系统自动记录原班级信息
- **批量修改组合**：将选中学生的选科组合统一变更为目标组合
- **批量分班**：同时修改班级和选科组合，适用于分班场景

### 3.6 Excel 导入学生

1. 点击"导入学生"按钮
2. 点击"下载模板"按钮，获取标准 Excel 导入模板
3. 按模板格式填写学生信息（必填列：学号、姓名、班级；选填列：原班级、性别、组合、身份证号、入学年份）
4. 点击"选择文件"，上传填写好的 Excel 文件
5. 系统自动预览导入结果，显示新增学生数和已有学生数
6. 确认无误后点击"确认导入"

**注意事项**：

- 学号不能重复，重复的学号将被跳过
- 班级名称不存在时，系统会自动创建该班级
- 导入前请确保 Excel 文件格式正确，列名与模板一致

### 3.7 Excel 导出学生

点击"导出"按钮，系统将全部学生信息导出为 Excel 文件，包含学生基本信息及最近一次考试的成绩。

## 四、考试管理

点击左侧导航栏"考试管理"，进入考试管理页面。

### 4.1 创建考试

1. 点击"新增考试"按钮
2. 填写以下信息：
   - **考试名称**：必填，如"2026 届高三一模"
   - **考试日期**：必填，选择日期
   - **考试类型**：下拉选择，模考/月考/期中/期末
   - **考试对象**：下拉选择，高一/高二/高三/全部
   - **是否分班后考试**：勾选表示这是分班后的考试（排名时使用现班级）
3. 点击"保存"按钮

创建成功后，系统自动为该考试关联 9 个学科（语文、数学、英语、物理、化学、生物、政治、历史、地理），其中化学、生物、政治、地理四科标记为需要赋分。

### 4.2 查看考试详情

点击考试列表中某条考试的名称，进入考试详情页，可查看：

- 考试基本信息（名称、日期、类型、对象）
- 各科目满分及是否需要赋分
- 已录入成绩数量
- 分数段分布柱状图

### 4.3 删除考试

在考试列表中点击"删除"按钮，确认后删除该考试及其关联的全部科目和成绩数据。

**警告**：删除操作不可恢复，请谨慎操作。

### 4.4 清空所有数据

在考试管理页面点击"清空所有数据"按钮，系统将清除全部学生、考试、成绩数据，仅保留管理员账户和赋分规则。此操作不可恢复。

## 五、成绩管理

### 5.1 录入成绩

进入考试详情页，在成绩录入区域操作：

- **单条录入**：选择学生，输入分数，点击"提交"
- **批量录入**：选择科目，为该科目下的多个学生一次性录入分数

### 5.2 Excel 导入成绩

1. 在考试详情页点击"导入成绩"按钮
2. 点击"下载模板"，获取标准成绩导入模板（列：学号、姓名、语文、数学、英语、物理、化学、生物、政治、历史、地理）
3. 按模板填写成绩数据
4. 上传填写好的 Excel 文件
5. 系统按学号自动匹配学生并写入成绩

**注意事项**：

- 学号必须与已导入的学生学号一致
- 成绩导入后，之前的成绩计算结果会被重置，需要重新计算
- 缺考科目的单元格留空即可

### 5.3 导出成绩

在考试详情页点击"导出成绩"按钮，将该次考试全部学生的成绩汇总导出为 Excel 文件，包含学号、姓名、各科成绩、总分和排名。

## 六、成绩计算

点击左侧导航栏"成绩计算"，进入成绩计算页面。

### 6.1 执行成绩计算

1. 在页面顶部选择要计算的考试
2. 选择计算模式：
   - **不进行赋分**：仅计算原始总分和排名
   - **进行赋分**：同时执行赋分计算（需选择赋分模板）
3. 若选择赋分模式，从下拉框中选择一个赋分模板
4. 点击"计算"按钮
5. 系统自动执行以下流水线：
   - 原始总分计算（按选科组合的 6 科）
   - 选科组合自动检测
   - 赋分计算（如启用）
   - 班级排名和组合排名

计算完成后，页面展示结果表格，包含以下列：

| 列名 | 说明 |
|------|------|
| 学号 | 学生学号 |
| 姓名 | 学生姓名 |
| 班级 | 所在班级 |
| 组合 | 选科组合 |
| 语文~地理 | 9 科原始成绩 |
| 原始总分 | 按组合 6 科计算的原始总分 |
| 班级排名 | 原始总分在班级中的排名 |
| 组合排名 | 原始总分在同组合中的排名 |

启用赋分时，额外显示：

| 列名 | 说明 |
|------|------|
| 化学/生物/政治/地理等级 | 赋分科目所处等级（A/B/C/D/E） |
| 化学/生物/政治/地理赋分 | 赋分后分数 |
| 赋分总分 | 用赋分替换原始分后的总分 |
| 赋分班级排名 | 赋分总分在班级中的排名 |
| 赋分组合排名 | 赋分总分在同组合中的排名 |

### 6.2 按班级/组合筛选

计算结果表格上方提供筛选器：

- **按班级筛选**：选择特定班级，仅显示该班学生
- **按组合筛选**：选择特定组合，仅显示该组合学生

### 6.3 导出计算结果

点击"导出"按钮旁的下拉箭头，可选择：

- **导出全部**：导出全部学生的计算结果
- **分班级导出**：每个班级生成一个独立的 Sheet
- **分组合导出**：每个组合生成一个独立的 Sheet

### 6.4 导出赋分报告

启用赋分模式后，点击"导出赋分报告"按钮，系统生成各赋分科目的等级统计报告，包含每个等级的人数、占比、原始分区间、赋分区间、赋分均分等信息。

## 七、赋分模板管理

赋分模板用于定义各赋分科目各等级的人数占比，支持校本化配置。

### 7.1 创建赋分模板

1. 在成绩计算页面点击"新建模板"按钮
2. 填写模板名称（如"2026 届高三赋分模板"）
3. 分别设置四门赋分科目（化学、生物、政治、地理）的各等级占比：
   - A 等级占比（默认 15%）
   - B 等级占比（默认 35%）
   - C 等级占比（默认 35%）
   - D 等级占比（默认 13%）
   - E 等级占比（默认 2%）
4. 系统实时校验：每个科目的各等级占比之和必须等于 100%
5. 点击"保存"按钮

**说明**：

- 占比决定了该等级容纳的考生比例。例如 A=15% 表示排名前 15% 的考生进入 A 等级
- 不同科目的占比可以不同，以适应各科实际生源分布
- 建议根据学校近几次考试的排名分布来设定合理的占比

### 7.2 编辑赋分模板

1. 在模板下拉框中选择要编辑的模板
2. 点击"编辑"按钮
3. 修改等级占比后点击"保存"

### 7.3 删除赋分模板

选择模板后点击"删除"按钮，确认后删除。

## 八、统计分析

### 8.1 年级分析

点击左侧导航栏"统计分析 → 单次考试分析 → 年级分析"，首先选择要分析的考试。

年级分析页面包含以下板块：

**数据总览**：展示各科参考人数、均分、最高分、最低分、及格率、优秀率。

**各科均分对比**：柱状图展示各科均分，附带及格率和优秀率的对比柱状图。

**各组合数据总览**：按班级展示原始总分和赋分总分的均分、最高分、最低分、中位数。

**考试质量分析表**：按班级×组合×学科交叉统计最高分、最低分、班级前 10 名均分、后 10 名均分、班级均分、中位数。可一键导出为 Excel。

**成绩分段统计**：总分按 50 分一段、单科按 10 分一段，统计各段人数及占比，按班级分列。

**排名分布**：两个环形图分别展示年级前 N 名和后 N 名学生在各班的分布情况。默认前 30 名和后 30 名，可自定义人数。

**分布形态分析**：计算各科及总分的偏度和峰度，自动判断分布形态（正态/右偏/左偏、扁平/尖峰），以渐变色直方图展示分布情况。

**临界生统计**：统计各科在及格线附近（±10 分）的学生人数及占比。语文/数学/英语及格线为 90 分，其余科目为 60 分。

**一键导出**：页面各板块均提供"导出 Excel"按钮，导出的文件与网页展示格式一致。

### 8.2 班级分析

点击"统计分析 → 单次考试分析 → 班级分析"，选择考试后进入。

**班级对比雷达图**：以雷达图形式对比各班各科均分，每个多边形代表一个班级，可直观看出各班的学科优势和短板。

**各班均分柱状图**：柱状图对比各班的各科均分。

### 8.3 个人分析

点击"统计分析 → 单次考试分析 → 个人分析"，选择考试后进入。

**学生选择**：通过班级下拉框和学生搜索框选择目标学生。

**成绩详情**：展示该学生各科成绩、班级排名、年级排名。

**个人与班级对比雷达图**：以雷达图叠加展示该学生各科成绩与班级均分的对比，实线为个人成绩，虚线为班级均分。

**薄弱科目检测**：自动识别低于班级均分 10 分以上的科目，标记为薄弱科目。

### 8.4 多次考试分析

点击"统计分析 → 多次考试分析"，选择两次或多次考试进行对比分析。

**年级各科均分趋势**：折线图展示各科均分在多次考试间的变化趋势。

**总分均分对比**：柱状图对比各次考试的年级总分均分。

**及格率/优秀率趋势**：折线图展示各科及格率和优秀率的变化趋势，实线为及格率，虚线为优秀率。

### 8.5 个人追踪分析

点击"统计分析 → 个人追踪分析"，选择学生后查看其历次考试的纵向趋势。

**总分趋势**：平滑面积图展示学生历次考试的总分变化，可直观判断成绩走势。

**各科趋势**：多条折线图叠加展示该学生各科历次成绩的变化，支持图例滚动筛选。

## 九、常见问题

### 问：登录后页面空白或报错怎么办？

答：请确认系统服务是否正常启动（命令行窗口是否显示 `Application startup complete`）。若服务未启动，在项目目录下执行 `python run.py` 启动系统。

### 问：导入 Excel 时提示"学号不匹配"怎么办？

答：成绩导入时按学号匹配学生。请确保成绩 Excel 中的学号与之前导入学生时使用的学号完全一致。建议先在学生管理页面确认学生信息已正确导入。

### 问：赋分计算后部分学生的赋分为空？

答：赋分仅对化学、生物、政治、地理四门选考科目生效。语文、数学、英语、物理、历史不参与赋分。如果某学生未选考某门赋分科目（如"史政地"组合不包含化学），则该科无赋分结果。

### 问：计算结果与 Excel 手工计算不一致？

答：请检查以下几点：（1）学生的选科组合是否正确（系统会自动检测，但建议手动确认）；（2）赋分模板的等级占比设置是否正确；（3）是否有缺考学生（缺考科目不参与计算）。

### 问：如何备份数据？

答：系统的数据库为单个 SQLite 文件，位于项目目录下的 `data/scores.db`。直接复制该文件即可完成备份。恢复数据时，将备份文件覆盖原文件即可。

### 问：如何修改管理员密码？

答：目前系统暂不支持在线修改密码。如需修改，请联系系统管理员通过数据库工具直接修改。
